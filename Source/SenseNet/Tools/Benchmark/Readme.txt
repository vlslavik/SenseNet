Installation

Copy Benchmark.ashx to the website's directory.
Copy SenseNet.Benchmark.dll to the website's bin directory.
Copy to all instance if the tested site runs NLBS.

Testing

Configure the SenseNet.BenchmarkTool (i.e. edit the SenseNet.BenchmarkTool.exe.config).
Run the SenseNet.BenchmarkTool.exe (always only one instance).
Analyze logs after tool is finished.
